# 书斋 · 山月照书窗

A Li Bai–inspired personal library: moonlit space, slow motion, poetic bilingual copy.

## Develop

```bash
npm install
npm run dev
```

## Structure

- `/` Pavilion entrance
- `/library` 藏书 (Novel / Psychology / Chinese Culture / Notes)
- `/projects` 造物 (AI / Open Source)
- `/journal` 随笔
- `/about` 山中人
- `/contact` 鸿雁

Content and poetic copy live in `src/data/site.ts`.
